# City Twin (Next.js app)

This folder contains the Next.js app for the AI City Digital Twin project.

For the main project overview, see the repository root README: `../README.md`.

## Getting Started

Install dependencies and run the dev server:

```bash
npm ci
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## Deployment (Vercel)

The repository includes a `vercel.json` at the repo root that deploys this app from `city-twin/`.
